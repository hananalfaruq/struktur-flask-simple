# struktur-flask-simple
Struktur Flask Mahasiswa sederhana yang dibuat menggunakan Python

Proyek ini adalah aplikasi REST API sederhana menggunakan Flask dan PostgreSQL untuk mengelola data mahasiswa.
Aplikasi ini memiliki 4 endpoint utama: GET, POST, DETAIL (GET by NIM), dan DELETE.
